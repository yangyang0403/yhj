CLIENT

디지털 아카이브 J사

FUNCTION

  • 연구자용 도록과 일반 해설을 별도로 제공
  • 디지털 소장자료를 통한 원문 이미지 제공
  • 음성지원 콘텐츠, 가상 전시 등 조선통신사 기록물을 활용한 콘텐츠를 개발 및 제공
  • 유네스코 세계기록유산으로 등재된 조선통신사 기록물 정보 통합 검색

SERIVCES

  • 관리자 페이지 구현
  • 기록물 소장처 표기
  • 분류별 기록물 별도 필터링 기능
  • 기록물별 해설 제공
  • AR 가상전시관 구현
  • 기록물 전용 Viewer 구현

REQUEST

우리네 조선통신사 기록물의 역사적 가치와 우수성을 전하는 디지털 아카이브
국내·외에 분산된 유네스코 세계기록유산 등재 조선통신사 기록물의 통합 정보시스템을 구축하여 조선통신사 기록물의 역사적 가치와 우수성을 전파
조선통신사 기록물을 디지털로 간편하게 볼 수 있는 플랫폼
  • 관리자 페이지 구현
    도록 및 디지털소장자료를 직접 등록하고 수정, 관리
  • 기록물 소장처 표기
    기록물을 실제 소장하고있는 소장처 정보 구현(Footer)
  • 분류별 기록물 별도 필터링 기능
    등재구분별, 제작시기별, 소장국가별, 형태별, 소장처별
  • 기록물별 해설 제공
    전문가용 기록물(도록) 및 일반인 해설서 별도 제공
  • AR 가상전시관 구현
    실제 박물관에 방문하여 전시된 기록물을 읽는듯한 콘텐츠 경험 제공
  • 기록물 전용 Viewer 구현
    기록물, 디지털소장자료 등 각 자료를 온라인상에서 책자를 읽듯이 책장을 넘기며 볼 수 있는 전용 Viewer 제공

solution

  • 책자정보 등의 단편적인 내용으로만 접근할 수 있는 낯설고 어려운 조선통신사 기록물
  • 기록물별 다양한 매체형식의 서비스를 통하여 언제 어디서나 조선통신사 기록물을 접할 수 있음
  • 조선통신사 기록물을 일일이 원하는 조건으로 분류하여 원하는 내용만 확인하기 힘듬
  • 구분별(외교기록, 여정의 기록, 문화교류의 기록), 제작시기별, 형태별 등 원하는 내용의 조선통신사 기록물을 간편하게 모아서 열람 가능
  • 별도의 텍스트로만 접근이 가능하던 각종 기록물
  • 가상전시관/텍스트가 아닌 이미지로 구현된 전용 Viewer 등으로 실제 기록물을 읽는 듯이 열람 가능
