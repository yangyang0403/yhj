CLIENT

디지털 아카이브 J사

FUNCTION

  • 국가별 전시관 제작(조선관, 일본관)
  • 핫스팟 태그를 이용한 지정된 장소 이동 및 기록물 확대
  • 디지털 소장자료 원문이미지 제공
  • 음성지원 및 가상전시

SERIVCES

  • 가상 3D 전시관 Archinteriors 모델링
  • 360° Panorama image Rendering
  • 3D Vista Virtual Tour
  • 태그 지정
  • 기록물 전용 Viewer 구현
  • 홈페이지 기능 추가

REQUEST

우리네 조선통신사 기록물의 역사적 가치와 우수성을 전하는 디지털 아카이브
부산의 유일한 유네스코 세계기록유산인 조선통신사 기록물을 가상으로 구축한 3D 전시관에서 온라인으로 둘러볼 수 있는 투어 기능 제공
온라인 가상 3D 전시 프로그램
  • 가상 3D 전시관 Archinteriors 모델링
    조선관 및 일본관으로 분류
  • 360° Panorama image Rendering
    모델링된 가상 전시관을 파노라마 이미지로 렌더링
  • 3D Vista Virtual Tour
    렌더링된 파노라마 이미지를 Stitching 작업을 통해 가상 3D 공간 제작
  • 태그 지정
    각 기록물별 이동 기능 핫스팟 태그 구현
  • 기록물 전용 Viewer 구현
    열화된 낮은 해상도로 인한 투어 불편을 최소화하는 차원에서 전용 Viewer 제공 (기록물 클릭시 고해상도 원본 이미지 노출)
  • 홈페이지 기능 추가
    자사 홈페이지에 가상 전시관으로 분류하여 추가

solution

  • 홈페이지에 텍스트 설명문 및 기록물 이미지로만 구성된 한정적 구현
  • 3D 형태의 가상전시관을 구축하여 VR 투어 기능을 제공하고, 각 기록물별 위치 이동 태그지정 및 전용 Viewer 제공
