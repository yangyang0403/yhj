CLIENT

스틸가공 전문 업체 H사

FUNCTION

  • 바코드스캔을 통한 제품정보조회
  • 실시간 원/부자재 내역관리
  • 모바일 입·출고 프로그램

SERIVCES

  • PDA연동
  • 실시간 데이터 조회
  • 모바일 입·출고 시스템

REQUEST

바코드 스캔을 통한 간편한 입.츨고 시스템
거래명세표로 일일이 대조하던 방식에서 바코드 스캔을 통한 쉽고 간편한 입.출고 관리, 실시간으로 데이터 조회도 확인 가능
입고, 발주, 조회등 모바일로 관리하는 내손안의 작은 ERP
  • PDA연동
    산업용 PDA의 바코드스캔 기능을 안드로이드 S/W와 결합하여 현장에서 보다 빠르고 편리하게 작업하도록 함
  • 실시간 데이터 조회
    PDA 스크린을 통해 원자재 입출고현황, 산적 위치, 재고현황 확인
  • 모바일 입·출고 시스템
    원/부자재 입·출고에 대한 데이터베이스 작업을 PDA를 통해 현장에서 즉시 처리

solution

  • 원/부자재 입·출고 수량을 수기로 작성하여 관리함
  • 모바일 앱을 연동하여 실시간으로 원/부자재 입출고 내역 관리
  • 수기로 작성한 라벨을 부착해서 재공·재고 관리
  • 바코드를 이용하여 실시간으로 재공·재고의 산적위치 등 관리
  • Paper에 의한 출하관리로 정확한 출고내역 확인 불가
  • 제품 출하 전 재고의 실시간 관리
